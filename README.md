# USA Telegram Session Bot

এই বটটি USA Proxy দিয়ে টেলিগ্রাম ইউজার সেশন তৈরি করে।

## Features
- USA Proxy Connection
- 24/7 Online (GitHub)
- Secure Sessions
- User Database

## Commands
- `/start` - Start bot
- `/create` - Create session
- `/status` - Bot status
- `/help` - Help guide

## Deployment
Automatically deployed on GitHub Actions.
